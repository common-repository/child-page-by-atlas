Плагин вывода в сайдбаре дочерних страниц

== Description ==
Плагин создает виджет, который выводит дочерние страницы (если они есть) в сайдбаре у выбраной страницы в настройках виджет.

== Installation ==

1. Скопируйте папку плагина Child page by Atlas в /wp-content/plugins/.
2. Активируйте плагин через меню Плагины.
3. Это все.
